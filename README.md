# Fresh Forward - Frontend

## Tech Stack
- **React** - UI library
- **TypeScript** - typed JS
- **Vite** - build tool
- **Tailwind CSS** - styling
- **React Router** - page routing

## Workflow - READ THIS
**Do NOT push directly to main.** Every change goes through a branch + Pull Request.

```bash
git checkout main
git pull
git checkout -b yourname/description

# code

git add .
git commit -m "clear description of change"
git push -u origin yourname/description
```

Then open a PR on GitHub. Get it reviewed before merging into main.

## Setup
```bash
npm install
npm run dev
```
